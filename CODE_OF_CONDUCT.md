Code of Conduct

This project is committed to a respectful and welcoming environment.
All contributors must:

Treat others with respect and professionalism.

Avoid insulting, offensive, or harmful behaviour.

Use inclusive and friendly language.

Help create a positive and constructive discussion space.

Accept feedback gracefully and give helpful feedback to others.

Follow project guidelines and collaborate responsibly.

Thank you for helping build a healthy open-source environment.
